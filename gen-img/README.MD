# Gemini 2.0 Flash Hikaye ve Görsel Oluşturucu

Bu uygulama, Google'ın Gemini 2.0 Flash (Image Generation) Experimental modelini kullanarak hikayeler ve görseller oluşturan bir PHP web uygulamasıdır.

## Kurulum

1. Dosyaları web sunucunuza yükleyin.
2. Google AI Studio'dan bir API anahtarı alın.
3. `config.php` dosyasındaki `$apiKey` değişkenine API anahtarınızı girin.
4. Web tarayıcınızda uygulamayı açın.

## Gereksinimler

- PHP 7.4 veya daha yüksek
- Web sunucusu (Apache, Nginx vb.)
- Google AI Studio'dan alınmış API anahtarı

## Kullanım

1. Hikaye teması, karakterler, ortam ve diğer ayarları girin.
2. Görsel stilini seçin.
3. "Hikaye ve Görselleri Oluştur" düğmesine tıklayın.
4. Oluşturulan hikaye ve görselleri görüntüleyin.
5. İsterseniz hikayeyi yazdırabilir veya PDF olarak kaydedebilirsiniz.

## Notlar

- Bu uygulama, Google'ın Gemini 2.0 Flash API'sini kullanır.
- API'nin kullanımı, Google'ın kullanım şartlarına tabidir.
- Şu anda API entegrasyonu tamamlanmamıştır ve gerçek API'ye bağlanmıyor, örnek veriler gösteriliyor.
- API'nin gerçek entegrasyonu için, index.php dosyasındaki yorum satırlarını kaldırın ve kendi Gemini API anahtarınızı ekleyin.
*/
